Main Stack

The main stack (also called the call stack) is where JavaScript executes your code line by line synchronously. JavaScript is single-threaded, meaning: It has one main stack.

function a() {
  console.log("A");
  b(); }
function b() {
  console.log("B"); }
a();
console.log("C");

Side Stack

This isn’t literally a “stack”, but people often call it a side stack to mean the place where asynchronous tasks wait until the main stack is free.

What is Synchronous?

Synchronous code runs line by line, one after another each line must finish before the next one starts. It blocks the code execution until the current task is done. 

console.log("Start");
function add(a, b) {
  return a + b; }
let result = add(5, 10);
console.log(result);
console.log("End");

What is Asynchronous?

Asynchronous code allows tasks to run in the background, without blocking the rest of the code.
Other code can continue running while the async task finishes later. 

console.log("Start");
setTimeout(() => {
  console.log("Cooking done!");
}, 2000); // 2 seconds delay
console.log("End");

Blocking and Non-Blocking

Blocking means an operation or function pauses the entire program until it completes, while non-blocking allows the program to continue with other tasks while waiting for the operation to finish. Blocking is like synchrounous of a function and non-blocking is like as asynchronous.

Working of fetch()

The purpose of fetch in JavaScript is to make HTTP requests from your code, usually to get or send data to a server or API.
tfetch is used to connect to web services (like APIs), get data (like JSON format), or send data (like form submissions)
Syntax of fetch api is as follows: 

fetch(url, options)
  .then(response => response.json()) // parses response
  .then(data => {
    // do something with the data
  })
  .catch(error => {
    // handle errors like no internet or bad server response
  });

setTime()

Runs after a certain time. The time will given to it in milli seconds. The code will run after the given time complete.
It can accept two main things:
setTimeout(function, delay); 

setTimeout(() => {
  console.log("This runs
   after 2 seconds");
}, 2000); // 2000 ms = 2 seconds

setInterval

Runs code repeatedly after the X miliseconds, where x is the input of the miliseconds.
setInterval(function, interval);
The following code in this example should run after one second 

setInterval(() => {
  console.log("This runs every 1 second");
}, 1000); // Repeats every 1 second

replaceAll() & split()

replaceAll() are used to replace all of the things that were selected
split() method is used to split a string into an array based on a given separator (like space, comma, or any character). 

const str = "Hello World, Hello!";
console.log(str.replaceAll("Hello", "Hi")); // "Hi World, Hi!"
const str = "Hello,World,!";
console.log(str.split(",,,")); // ["Hello",,, "World",,, "!"]
