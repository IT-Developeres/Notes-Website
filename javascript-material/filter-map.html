Map method

Map in js arrays is used in place or loops whenever we need to use loops we use map method. map method is used to map every element of array. Map transform each element in an array and returns a new array with the transformated values. 

const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map
(function(num) {
return num * 2;
});
console.log(doubled); 
// [2, 4, 6, 8, 10]

Filter method

Filter is also like map type method, but the difference is that it contains on the items in an array where function give true value. 

const names = ["Ali", "Sara",
 "Ahmed", "John", "Ayesha"];
const aNames = names.filter
(function(name) {
 return name.startsWith("A");
});
console.log(aNames); 
// ["Ali", "Ahmed", "Ayesha"]

forEach method

forEach() lets you loop through each item in an array and do something with it like print, update, or call a function but it does NOT return a new array.

const names = ["Ali", "Sara", "John"];
names.forEach(function(name) {
  console.log(name);
});
const numbers = [10, 20, 30];
numbers.forEach((num, index) => {
  console.log(`Index ${index}: ${num}`);
});
