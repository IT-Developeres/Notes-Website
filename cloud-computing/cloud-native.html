What is Cloud?

Cloud computing refers to the delivery of computing services like servers, storage, databases, and networking over the internet to offer faster innovation, flexible resources, and economies of scale.
In simple words cloud is a server which is accessible on the server via internet

Types of Cloud

There are three types of Clouds in native computing

  • Public Cloud
  • Private Cloud
  • Hybrid Cloud

Public Cloud

Which is publically accessible over the internet. AWS, Amazon and Google cloud are examples of public clouds.

Private Cloud

Which is publically accessible over the internet. Banks or departments are the basic examples.

Hybrid Cloud

Hybrid Cloud are the combination of public cloud or private cloud.
